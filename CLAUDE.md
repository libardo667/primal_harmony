## Architecture

### Data-Driven Design

- **No hardcoded stats.** All game data lives in `data/` as JSON or `.tres`.
- World graph: `data/world_connections.json` (518 maps, generated by `tools/build_world_connections.py`)
- NPC placement: `data/npc_placements.json` (keyed by pokeemerald map ID, e.g. `"MAP_LITTLEROOT_TOWN"`)
- Pok√©mon/moves/items in `data/pokemon/`, `data/moves/`, `data/items/` (CSV-based + per-species JSON)
- Zone definitions: `data/zones/`
- Each data subdirectory has a `_schema.json` ‚Äî validate against it when modifying data files
- `data/npc_lore.json` ‚Äî writer's reference (428 maps, 2776 NPCs, 1048 with dialogue) ‚Äî **NOT loaded at runtime**

### DataManager API

```gdscript
DataManager.get_pokemon(species_id: String) -> PokemonData   # RefCounted, NOT Dictionary
# PokemonData fields: id, name, national_dex, types[], base_stats{}, abilities[],
#   learnset[], sprites{front/back/icon}, is_corrupted, corruption_zone, base_species

DataManager.get_pokemon_display_name(species_id: String) -> String
# Falls back: PokemonData.name ‚Üí all_pokemon.csv dex lookup ‚Üí capitalize(species_id)

DataManager.get_move(move_id: String) -> MoveData
# MoveData fields: name, type, category (physical/special/status), power, accuracy, pp, effect, flags[]
# Move category is per-move PSS split ‚Äî NOT type-based. Fairy type exists in type chart.
```

`DataManager` level-filters learnset in `BattleManager.build_pokemon_state()` (only moves with `level <= current_level`).

### Signal Flow

- `Player.gd` emits `stepped_on_tile(Vector2i)` every 16px step and `interact_pressed(facing_tile: Vector2i)`
- `MainGame.gd` listens to both: checks `WorldConnections` for warps/edge transitions, calls `NPCSpawner.try_interact()`
- `NPCSpawner.spawn_npcs_for_map(map_id, map_node)` called after each map load; NPCs are freed with the map
- `ZoneVisuals.setup_for_map()` applies shaders after each map load
- Encounter chain: `Player._try_encounter_step()` ‚Üí `EncounterManager.try_encounter(zone_id)` ‚Üí `TriageCatch` intercepts (corrupted species + infested/partial zone ‚Üí `TriageChoiceMenu`; else ‚Üí `BattleManager.start_wild_battle()`)
- **Signals over direct calls** for cross-system communication

### Z-Layer Rendering Order

| Layer | Value | Notes |
|---|---|---|
| `TileMapTerrain` | z=0 | Always below characters (COVERED tops on layer 2) |
| Player + NPCs | `int(global_position.y / 16) + 100` | Y-sorted each physics frame (~100-200) |
| `TileMapDecoration` | z=500 | Set at runtime by `MainGame._load_map()` ‚Äî NORMAL/SPLIT tops (roofs, canopy) always above player |
| Fog overlay `CanvasLayer` | layer=10 | |
| `DialogueManager` `CanvasLayer` | layer=100 | |
| `TriageChoiceMenu` `CanvasLayer` | layer=200 | |

### Map Scene Structure

Each of 440 map scenes (in `maps/hoenn/`) has exactly:
- `TileMapTerrain` ‚Äî base layer (z=0)
- `TileMapDecoration` ‚Äî NORMAL/SPLIT top tiles (z set to 500 at runtime ‚Äî always above player)
- `TileMapCollision` ‚Äî hidden physics layer
- Root Node2D metadata: `zone_id`, `tile_size`, `map_dimensions_px`

Maps are batch-painted by `tools/batch_paint_maps.gd` from pokeemerald `layouts.json`. Naming: CamelCase ‚Üí snake_case; Route* ‚Üí `routes/`; underscore-in-name ‚Üí `interiors/`; else ‚Üí `cities/`.

### Collision Layer Contracts (bitmask values in .tscn)

| Purpose | `collision_layer` value |
|---|---|
| World collision (terrain) | `1` |
| Warp points | `2` |
| Encounter zones | `4` |
| NPC interaction | `256` |

Godot editor "Layer N" ‚âÝ bitmask value. Default is `1` ‚Äî **always set explicitly** on warp and encounter Area2Ds.

### Sprite Geometry

Player and NPC sprites: 16√ó32 px. `AnimatedSprite2D` at offset `(0, -8)` ‚Äî feet at tile bottom, head extends above.
- `CollisionShape2D` at `(0, 4)` size `14√ó8` (feet zone only)
- Y-sort: `z_index = int(global_position.y / 16) + 1` ‚Äî set every physics frame
- Set `npc.position` **before** `add_child()` so `_ready()` captures correct `_origin`
- GBA palette index 0 (mint green RGB 115,197,164) is transparent ‚Äî all 137 sprites corrected via `tools/fix_sprite_transparency.py`

### Tileset Animation Pipeline

1. **Python** (`tools/export_tileset_anim_frames.py`): parse `pokeemerald/src/tileset_anims.c` ‚Üí atlas PNGs + JSON ‚Üí `assets/tilesets/anim/`
2. **EditorScript** (`tools/build_anim_tilesets.gd`): JSON ‚Üí `assets/tilesets/{ts}_anim_bottom.tres` + `_top.tres`
3. **Runtime** (`AnimTileLoader.gd`): after map load, routes animated cells to anim sources; Godot animates natively

GBA timing: general anims `timer % 16` at 60fps = 3.75 fps; building anims `timer % 8` = 7.5 fps.

### TOZ Visual System

- Tilemap shader: `shaders/toz_tilemap.gdshader` ‚Äî `flash_impulse`-based; set to `1.0` every 4 steps when infested
- Fog overlay: `shaders/fog_overlay.gdshader` ‚Äî greyscale PNG + `fog_tint` per zone (e.g. The Murk: `(0.28, 0.75, 0.10)` toxic green)
- EHI thresholds: infested ‚â§35, partial 35‚Äì67, restored >67 ‚Äî restored clears all materials

## GDScript Conventions

- **Use static typing.** Declare all variable and return types explicitly.
- **Signals over direct calls** for cross-system communication.
- No hardcoded stats or move data ‚Äî always read from `data/`.
- Data objects are `RefCounted` subclasses (`PokemonData`, `MoveData`, etc.) ‚Äî never raw `Dictionary`.
